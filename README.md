–ù–∏–∂–µ –∫–æ–¥, –∫–æ—Ç–æ—Ä—ã–µ –Ω–µ –ø–æ–º–µ—á–∞–µ—Ç —Ç–µ —Å–∏–º–≤–æ–ª—ã,–∫–æ—Ç–æ—Ä—ã–µ —É–∫–∞–∑–∞–Ω—ã,—Ç–∞–∫ –∂–µ –∏—Ö –º–æ–∂–Ω–æ –ø–æ–ø–æ–ª–Ω—è—Ç—å
```js
  const regexp = /[^\w\s'",\\.:;?!@#$%^&*+-={}\[\]()<>]/gi;
  result = text.replace(/\>/g, str => '&#x3E');
  result = result.replace(/\</g, str => '&#x3C');
  result = result.replace(regexp, symbol => '<mark>' + symbol + '</mark>');
```
